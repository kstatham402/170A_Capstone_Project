Elastic Net Logistic Regression - Identifing Cardiovascular Risk Factors in Prostate Cancer Patients on NHT¶

This notebook identifies baseline clinical factors associated with increased cardiovascular risk in prostate cancer patients on novel hormonal therapy (NHT). Cardiovascular risk is defined as a binary outcome ("at risk" vs. "no risk") based on new initiation of antihypertensive, lipid-lowering, or diabetes medications following NHT start.

We used an elastic net-penalized logistic regression with 5-fold cross-validation for variable selection across a broad set of clinical features, including demographics, comorbidities, baseline medications, tests, and referrals. Features selected by the elastic net are then passed to a standard logistic regression model to obtain odds ratios, confidence intervals, and p-values. Results are visualized as stratified bar charts comparing risk status across key predictors.

preprocessor = ColumnTransformer(
    transformers=[
        ("cont", Pipeline([
            ("impute", SimpleImputer(strategy="median")),
            ("scale", StandardScaler()),
        ]), continuous_features),

        ("binary", Pipeline([
            ("impute", SimpleImputer(strategy="most_frequent")),
        ]), binary_features),

        ("encoded", Pipeline([
            ("impute", SimpleImputer(strategy="most_frequent")),
            ("onehot", OneHotEncoder(drop="first", sparse_output=False, handle_unknown="ignore")),
        ]), encoded_features),
    ],
    remainder="drop",
)

X_processed = preprocessor.fit_transform(X_df)

Health history of smoking is suggestive but not significant.

Visualize Results¶

In [22]:
bp_labels = {1: 'Prior BP Meds', 0: 'No Prior BP Meds'}
df['bp_meds_label'] = df['bp_meds_prior'].map(bp_labels)

risk_labels = {0: 'Low', 1: 'High'}
df['risk_label'] = df['at_risk'].map(risk_labels)

risk_groups = ['Low', 'High']
colors = ['#378ADD', '#E24B4A']

y_cats = ['Prior BP Meds', 'No Prior BP Meds']
y_pos = [0, 0.45]

prop_data = {}
for cat in y_cats:
    subset = df[df['bp_meds_label'] == cat]
    total = len(subset)
    prop_data[cat] = {risk: (subset['risk_label'] == risk).sum() / total * 100
                      for risk in risk_groups}

fig, ax = plt.subplots(figsize=(7, 3))

lefts = np.zeros(len(y_cats))
for i, risk in enumerate(risk_groups):
    values = [prop_data[cat][risk] for cat in y_cats]
    bars = ax.barh(y_pos, values, left=lefts, color=colors[i],
                   alpha=0.85, label=str(risk), height=0.35)

    for bar, val in zip(bars, values):
        if val > 5:
            ax.text(bar.get_x() + bar.get_width() / 2,
                    bar.get_y() + bar.get_height() / 2,
                    f'{val:.1f}%', ha='center', va='center',
                    fontsize=13, color='white', fontweight='bold')
    lefts += values

ax.set_xlim(0, 100)
ax.set_yticks(y_pos)
ax.set_yticklabels(['Prior\nBP Meds', 'No Prior\nBP Meds'], fontsize=13)
ax.xaxis.set_major_formatter(plt.FuncFormatter(lambda x, _: f'{x:.0f}%'))
ax.tick_params(axis='x', labelsize=13)
ax.set_xlabel('Proportion of Patients', fontsize=14, labelpad=8)
ax.set_title('Risk Status by Prior Blood Pressure Meds', fontsize=16,
             fontweight='bold', pad=12)

ax.spines['top'].set_visible(False)
ax.spines['right'].set_visible(False)
ax.grid(axis='x', linestyle='--', alpha=0.4)
ax.legend(title='Risk Status', fontsize=12, title_fontsize=13,
          frameon=False, bbox_to_anchor=(1.01, 1), loc='upper left')

plt.tight_layout()
plt.show()
